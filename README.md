# Employee Management System

## Overview
A backend-focused system designed to automate employee leave requests, approvals, and administrative workflows for small businesses.

## Features
- Employee records management
- Leave request and approval workflow
- Timesheet tracking
- Admin feedback automation

## Tech Stack
- Java
- Spring Boot
- REST APIs
- MySQL (or Supabase for others)
- Next.js

## Why I Built This
Manual leave processes were slow and inconsistent in my workplace. This system improves efficiency and visibility.

In active development
